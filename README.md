# 🛒 Customer Shopping Behavior Analysis Dashboard

## 📌 Project Overview
This project analyzes customer shopping behavior using a dashboard.  
It helps to understand customer patterns, purchasing trends, and key business insights using visualizations.

---

## 🎯 Objective
- Analyze customer data to identify trends  
- Understand customer preferences  
- Support better business decision-making  

---

## 🛠️ Tools & Technologies Used
- Power BI / Tableau (for dashboard)
- Excel / CSV dataset
- Data Visualization techniques  

---

## 📊 Features
- Total Customers Overview  
- Average Purchase Amount  
- Customer Review Ratings  
- Revenue by Category  
- Sales by Category  
- Revenue & Sales by Age Group  
- Interactive Filters (Gender, Category, Shipping Type, Subscription Status)  

---

## 📈 Key Insights
- Most customers are non-subscribed  
- Accessories category generates highest revenue  
- Adult age group contributes the most sales  
- Filters allow dynamic analysis
